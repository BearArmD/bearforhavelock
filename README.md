# Bear for Havelock

Campaign website for Harlan "Bear" McGillem, candidate for **Havelock City Commissioner** (November 2026).

**Live site:** https://bearforhavelock.com

## About

A lifelong Havelock native running on three plain commitments: enforce the ordinances already on the books, get commercial truck traffic off residential roads, and answer every constituent. Built and maintained by the candidate.

## How it works

- A single self-contained `index.html` — all styles, scripts, and images are embedded directly in the file. No build step, no framework, no dependencies.
- Hosted free on GitHub Pages.
- The contact form is handled by [Web3Forms](https://web3forms.com) (free tier); submissions are emailed to the campaign inbox.
- `og.png` is the social-share preview image referenced by the page's Open Graph / Twitter Card tags.

## Files

| File | Purpose |
| --- | --- |
| `index.html` | The entire website |
| `og.png` | Social share card (1200×630) |

## Updating the site

Edit `index.html`, commit, and GitHub Pages redeploys automatically within about a minute.

## Cost

$0. No funds were raised or spent to produce this site.
